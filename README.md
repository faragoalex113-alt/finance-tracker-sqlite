# 💰 Finance Tracker (SQLite)

Simple finance tracker built with Python and SQLite.

## 🚀 Features

* Add income and expenses
* Store data in SQLite database
* List all transactions

## 🛠 Tech Stack

* Python
* SQLite

## ▶️ How to run

```bash
python main.py
```
